Breakpoints

Because screen size shouldn't matter!

What brakpoints are?

In old times there was only computers screens where we could surf on the Internet. But that times are long gone and we have great variety of devices we can use now. No matter if it is small smartphone, or big tv screen, content of web sites should presents perfectly!

To achive this we have to redesign some page elements like font size or section layout, depends on screen size. Breakpoints are hardcoded display widths in our stylesheet which are compared to user screen width and then we can serve best site version for each user device.
